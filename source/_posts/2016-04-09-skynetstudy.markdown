安装
===
linux环境配置
1. 安装应用，需要修改成国内的源，比如阿里云
2. ```sudo apt-get install git```
运行例子官方文档缺少内容
1. ```sudo apt-get install autoconf```
2. ```apt-get install libreadline-dev```
3. 进入```./3rd/lua/lua```目录，```make linux```
